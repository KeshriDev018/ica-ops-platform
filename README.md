# Indian Chess Academy - Frontend

Frontend application for Indian Chess Academy Operations Platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Design System

- **Colors**: Navy Blue (#003366), Orange (#FC8A24), Cream (#FFFEF3), Olive Green (#6B8E23)
- **Fonts**: Figtree (primary), Bodoni Moda (secondary)
- **Style**: Clean grid, soft shadows, rounded cards (12px border radius)

## Project Structure

```
src/
├── components/
│   └── common/        # Reusable UI components
├── pages/
│   └── public/        # Public pages (Landing, Book Demo, etc.)
├── styles/            # Global styles and Tailwind config
└── App.jsx           # Main app component
```
